The ClientMessages class is a component of the Multiplayer extension project. This classes purpose is to contain the client messages data classes that the server will use to send data to each of the clients. For this recipe, the ClientMessages class contains only one message which stores an ID, x and y touch coordinates, and a color ID that will ultimately be read by the client in order draw a point on the receiving client's display.

Installation:
1. Include this class in the Multiplayer example project.
